# ADR-0002｜证据优先的离线管线与人工发布门禁

状态：已采纳  
日期：2026-07-22

## 决策

1. 播放前完成解析、规划和审核；播放期间不让模型即时决定弹点。
2. ASR/OCR/关键帧生成时间化 evidence；模型语义项必须引用 evidenceId。
3. MiniMax/豆包通过同一结构客户端可替换；结构输出经 Zod 校验。
4. 确定性 Planner 负责数量、间距、时长、画面负荷和财经禁词。
5. 自动管线只能生成 DraftExperience，人工审核与 publish 是独立动作。

## 原因

- 财经视频中的时间码、数字和方向不能由不可复现的在线生成结果控制。
- 用户观看时需要低延迟、离线兜底和一致的触点行为。
- 人工审核能够把版权、内容正确性、安全措辞和交互负荷放在同一门禁。

## 暂不解决

P0 未实现审核台与持久化 publish API；因此真实视频即使分析成功也仍是 draft，不会进入播放器。
