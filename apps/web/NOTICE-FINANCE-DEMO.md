# 财经推演工程演示说明

本仓库中的“财经推演 / 财包知识触点”能力仅用于项目内部评审及公开的非商业工程演示。

- 演示入口：`/?demo=finance-fed`
- 当前本地合成视频仅用于验证时间轴、连续播放和轻交互机制，不是已审核的财经内容，也不包含真实财经叙事。
- 对外发布前，必须由项目组替换为具备授权的视频、最终字幕、真实时间码及审核后的证据 ID。
- 财包只解释概念、条件和因果机制，不提供买卖建议、仓位、目标价或收益承诺。
- 演示不会保存原始语音，也不会推断用户财富状况、风险偏好或投资能力。

原仿真底座的开源许可与素材权利边界仍按仓库原文件执行；新增财包素材默认仅限本项目获授权范围内使用。
